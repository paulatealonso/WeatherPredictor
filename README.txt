## WeatherPredictor

## Description
WeatherPredictor is a Java application that allows users to get simulated weather forecasts for a variety of cities. The graphical interface is built using Swing and the project is designed to be simple and educational.

## Features
- Simulated weather forecasts for multiple global and local cities.
- Simple and friendly graphical user interface.

## Technologies Used
- Java
- Swing for the graphical user interface

## Installation
To run WeatherPredictor, you need to have Java installed on your machine. Clone this repository using the following command:

git clone https://github.com/tu_usuario/WeatherPredictor.git


Navigate to the project folder and run the `WeatherFrame.java` file using:

cd WeatherPredictor
java WeatherFrame.java



## Usage
Once the application is running, simply type the name of a city in the text field and click “Predict Weather” to view the forecast.
